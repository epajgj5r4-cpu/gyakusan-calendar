# 逆算カレンダー

「いつまでに何を成し遂げるか」を設定し、**ゴール日までの残り日数を毎日カウントダウン形式で確認しながら、その日の一歩を記録する目標管理アプリ**。
ToDoを細かく管理するのではなく、「あと何日しかない」と毎日意識できることに価値を置いています。

## 画面（3タブ構成）
- **今日**: 目標名・「あと◯日」・今日の日付・今日の一歩の入力／保存。最もシンプルで毎日開くメイン画面。
- **記録**: 保存した「今日の一歩」を日付ごとに新しい順で一覧表示。
- **目標設定**: 目標名・開始日・ゴール日の設定／編集／リセット。

## 特徴
- ライトテーマ＋淡いブルー/パープルのグラデーション、マスコット入りの親しみやすいUI
- **「あと◯日」を最も大きく**表示（グラデーション数字）
- その日の一歩を日付ごとに記録（同じ日は上書き保存）
- 「記録」タブは検索つきのタイムライン表示
- 「目標設定」タブは入力に応じて**期間プレビュー**をリアルタイム表示
- 「目標設定」タブで **If-Then ルール**（もし〇〇したら、△△する）を設定でき、「今日」タブに表示
- データはブラウザの `localStorage` に保存（サーバー不要・オフライン動作）

## 画像アセット
- マスコット（透過PNG）: `mascot-run.png`（今日）/ `mascot-write.png`（記録）/ `mascot-flag.png`（目標設定）
- アプリアイコン: `app-icon-512.png`（PWA / ホーム画面用）
- 元データ: `キャラ/` フォルダの3枚、`アイコン.png`

差し替える場合は同名で置き換えるか、`index.html` / `manifest.webmanifest` 内の参照を変更してください。背景付き画像を使う場合は透過処理が必要です。

## スマホで使う
スマホ向けに最適化済みです（下部タブ・安全領域対応・タップしやすいボタン）。
**ホーム画面に追加**すると、アドレスバーのないフルスクリーンのアプリとして起動できます。

- iPhone (Safari): 共有ボタン → 「ホーム画面に追加」
- Android (Chrome): メニュー → 「ホーム画面に追加」

## 使い方（ローカル）
`index.html` をブラウザで開くだけで動きます。

```
逆算カレンダー/
├── index.html   # アプリ本体
├── style.css    # スタイル・焦らせる演出
├── app.js       # ロジック（カウントダウン / localStorage / タスク・メモ）
├── 仕様書.md     # 仕様書
└── README.md
```

## デプロイ（GitHub → Vercel）
静的サイトのためビルド設定は不要です。

1. このリポジトリを GitHub に push
2. Vercel で「New Project」→ 対象リポジトリを Import
3. Framework Preset: **Other**（ビルドコマンド・出力ディレクトリは空のまま）
4. Deploy

以降、GitHub に push するたび Vercel が自動でデプロイします。

## 注意
`localStorage` のデータは閲覧した端末・ブラウザ内にのみ保存され、端末をまたいだ共有・同期はされません。
